遊戲介紹:
這一款橫向卷軸恐怖解謎劇情遊戲，玩家扮演著昏倒在地下室的學生，開始思考自己身處在哪裡，開始探索周圍事物，並且離開這裡。
操作說明:按A向前 按D向後 按Space跳躍 按Z起身(不在遊戲內放遊戲說明是為了希望讓玩家自行發現如何起身和移動)
Code:
Assets/Scenes/底下分成許多GameScene和Dialogue和MainMenu 代表每一個場景的資料夾
裡面包含每個場景所使用的圖檔、prefab、Scripts、Inventory items等等(有些共用的Script可能就會放在其他資料夾)

Assets/Bolchie/人物相關的圖片動作prefab等等
Assets/底下散落的Scripts則是人物動作
Assets/Inventory/物品欄相關的東西
